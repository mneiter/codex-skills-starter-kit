# Project Skills

The `.codex/skills/project/` area is reserved for repository-specific skills that should not become part of the reusable starter kit.

## Rules

- keep project skills specific to one adopting repository
- register project skills in the root `.codex/skills/skills-manifest.md` and `.codex/skills/skills-index.md`
- continue using the canonical layer tokens `orchestration`, `guardrails`, and `atomic`
- do not move reusable generic guidance here just to avoid updating the base layer or plugins
