# v1.1.0 - Next Iteration

## Summary

The next development cycle after the v1.0.0 baseline.

## Goals

- Skill validation tooling

- Plugin architecture

## Migration Note

This iteration replaces the old `packs/` extension model with `.codex/skills/plugins/`, moves canonical base layers under `.codex/skills/base/`, and reserves `.codex/skills/project/` for repository-specific skills.
